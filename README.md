# Упрощённая структура проекта

- Все исходные SCSS-файлы теперь в `assets/scss/`
- Все изображения в `assets/img/`
- Скомпилированный CSS будет в `dist/`
- Оставлены только нужные HTML-файлы

## Как собирать стили

1. Редактируйте SCSS в `assets/scss/`
2. Компилируйте в `dist/style.css`

## Структура

- assets/
  - img/
  - scss/
- dist/
  - style.css
- index.html
- center.html
- package.json
- ...
